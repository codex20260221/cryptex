# Security Policy

## Reporting a vulnerability

Please report vulnerabilities privately by emailing **security@michaelmawhinney.com** with:

- affected package/version,
- reproduction steps or proof of concept,
- impact assessment,
- suggested remediation (if available).

Please do **not** open public issues for undisclosed vulnerabilities.

## Response expectations

- Initial acknowledgment target: within 5 business days.
- Triage and severity assessment: as soon as practical.
- Coordinated disclosure preferred after a fix is available.

## Scope

This policy applies to this repository and published package versions.
